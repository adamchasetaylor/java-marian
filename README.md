1. Install SDKMAN 
https://sdkman.io/install

2. Install Java
`sdk install java 11.0.11.hs-adpt`

3. Install Maven
`sdk install maven 3.8.3`

4. Make sure you are using this version of Java
`sdk use java 11.0.11.hs-adpt`

4. Setup ENV
`sdk env init`

5. Run
`./run.sh`

6. Check your Spark application is running
http://localhost:4567/
